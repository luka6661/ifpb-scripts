Tema 3: Gerenciamento de Arquivos

Escrever um clone do “Windows Explorer” para a linha de comando;
Deve exibir arquivos e pastas;
Deve permitir Navegação por diretórios;
Deve possibilitar filtros de busca no diretório atual;
Deve possibilitar listagem de detalhes;
Deve possibilitar ordenação por nome, por grupo dono, por extensão;
Deve possibilitar a pré-visualização dos arquivos.
O que mais a criatividade permitir.
